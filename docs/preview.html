1. Main architecture evolution

V3-V6: VLM explanation consistency 중심

설명 일관성은 직관적으로 매력적이었지만 detector localization utility와 안정적으로 연결되지 않았다.

V7-V8: DINO visual diversity와 NEU-only 정리

DINO diversity는 consistency보다 낫지만 Random을 안정적으로 이기지는 못했다. Random baseline이 강하다는 사실이 명확해졌다.

V9: detector-aware + instance-rich pivot

Round0 detector로 pool을 score하고 instance-rich 샘플을 고르기 시작했다. 그러나 pseudo-instance weight 0.40은 recall 저하와 보수적 detector를 만들었다.

V10b: uncertainty + DINO 강화, instance weight 축소

V10b는 detector uncertainty 0.25, DINO distance 0.35, class deficit 0.15, pseudo-instance 0.25로 고정되었다.

Next: frozen multiseed validation

Seed43-46에서 Random vs V10b one-cycle을 비교한다. 결과를 본 뒤 weight를 다시 조정하지 않는다.
