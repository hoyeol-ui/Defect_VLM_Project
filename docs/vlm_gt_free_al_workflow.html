3. Pool scoring & selection

Frozen V10b score

S(x)=0.25U(x)+0.35D(x)+0.15B(x)+0.25I(x)
  • U: detector uncertainty
  • D: DINO visual distance/diversity
  • B: predicted class deficit
  • I: pseudo instance count score
